## Question
Given a column title as appear in an Excel sheet, return its corresponding column number.

For example:
```
    A -> 1
    B -> 2
    C -> 3
    ...
    Z -> 26
    AA -> 27
    AB -> 28 
    ...
```

# Example1:
```
Input: "A"
Output: 1
```
# Example2:
```
Input: "AB"
Output: 28
```

# Example3:
```
Input: "ZY"
Output: 701
```
