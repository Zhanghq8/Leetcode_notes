## Question
Remove the minimum number of invalid parentheses in order to make the input string valid. Return all possible results.

# Example1:
```
Input: "()())()"
Output: ["()()()", "(())()"]
```
# Example2:
```
Input: "(a)())()"
Output: ["(a)()()", "(a())()"]
```
# Example3:
```
Input: ")("
Output: [""]
```

# Note:
- The input string may contain letters other than the parentheses ( and ).
